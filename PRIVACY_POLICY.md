## 繁體中文版

### Tab Session Lite 隱私權政策

感謝您使用 **Tab Session Lite**。我們非常重視您的隱私。本隱私權政策說明我們如何處理您的資料。

#### 1. 資料收集

**Tab Session Lite 不會收集任何個人資料。**

本擴充功能僅在您的裝置本地儲存以下資訊：

-   分頁標題
-   分頁網址 (URL)
-   分頁圖示 (Favicon)
-   視窗位置與大小
-   分頁群組資訊（名稱、顏色）

#### 2. 資料儲存

所有資料**僅保存在您的本地裝置**中，使用 Chrome 瀏覽器提供的 `chrome.storage.local` API。

-   ✅ 資料儲存在您的電腦上
-   ❌ 資料**不會**上傳至任何遠端伺服器
-   ❌ 資料**不會**與任何第三方分享
-   ❌ 本擴充功能**不會**發送任何網路請求

#### 3. 資料安全

由於所有資料都保存在本地，資料安全取決於您裝置的安全性。我們建議：

-   保持作業系統和瀏覽器更新
-   使用強密碼保護您的電腦帳戶
-   注意不要在公用電腦上保存敏感的分頁 Session

#### 4. 第三方服務

**Tab Session Lite 不使用任何第三方服務**，包括但不限於：

-   分析工具（如 Google Analytics）
-   廣告服務
-   使用者追蹤服務
-   遠端資料庫

#### 5. 權限說明

本擴充功能申請以下 Chrome 權限：

| 權限             | 用途                             |
| ---------------- | -------------------------------- |
| `tabs`           | 讀取分頁標題、網址以保存 Session |
| `tabGroups`      | 讀取和恢復分頁群組資訊           |
| `storage`        | 在本地儲存 Session 資料          |
| `system.display` | 取得螢幕訊息以正確恢復視窗位置   |

這些權限僅用於實現核心功能，不會用於任何其他目的。

#### 6. 兒童隱私

本擴充功能不針對 13 歲以下兒童，也不會有意收集任何兒童的個人資訊。

#### 7. 政策更新

如果我們更新本隱私權政策，將在此頁面發布新版本並更新「最後更新日期」。

#### 8. 聯絡方式

如果您對本隱私權政策有任何疑問，請透過以下方式聯絡我們：

-   GitHub Issues: [https://github.com/ben0588]

## English Version

### Tab Session Lite Privacy Policy

Thank you for using **Tab Session Lite**. We take your privacy seriously. This Privacy Policy explains how we handle your data.

#### 1. Data Collection

**Tab Session Lite does not collect any personal data.**

This extension only stores the following information locally on your device:

-   Tab titles
-   Tab URLs
-   Tab favicons
-   Window positions and sizes
-   Tab group information (names, colors)

#### 2. Data Storage

All data is stored **exclusively on your local device** using Chrome's `chrome.storage.local` API.

-   ✅ Data is stored on your computer
-   ❌ Data is **NOT** uploaded to any remote servers
-   ❌ Data is **NOT** shared with any third parties
-   ❌ This extension does **NOT** make any network requests

#### 3. Data Security

Since all data is stored locally, data security depends on the security of your device. We recommend:

-   Keeping your operating system and browser up to date
-   Using strong passwords to protect your computer accounts
-   Being cautious about saving sensitive tab sessions on public computers

#### 4. Third-Party Services

**Tab Session Lite does not use any third-party services**, including but not limited to:

-   Analytics tools (such as Google Analytics)
-   Advertising services
-   User tracking services
-   Remote databases

#### 5. Permissions Explained

This extension requests the following Chrome permissions:

| Permission       | Purpose                                                        |
| ---------------- | -------------------------------------------------------------- |
| `tabs`           | Read tab titles and URLs to save sessions                      |
| `tabGroups`      | Read and restore tab group information                         |
| `storage`        | Store session data locally                                     |
| `system.display` | Obtain screen information to correctly restore window position |

These permissions are used solely to implement core functionality and will not be used for any other purpose.

#### 6. Children's Privacy

This extension is not directed at children under 13 years of age, and we do not knowingly collect any personal information from children.

#### 7. Policy Updates

If we update this Privacy Policy, we will post the new version on this page and update the "Last Updated" date.

#### 8. Contact Us

If you have any questions about this Privacy Policy, please contact us through:

-   GitHub Issues: [https://github.com/ben0588]
-   Email: [energy9527z@gmail.com]

### Tab Session Lite プライバシーポリシー

**Tab Session Lite** をご利用いただきありがとうございます。私たちはお客様のプライバシーを非常に重視しています。このプライバシーポリシーでは、お客様のデータの取り扱いについて説明します。

#### 1. データ収集

**Tab Session Lite は個人データを一切収集しません。**

この拡張機能は、以下の情報をお客様のデバイスにローカルで保存するのみです：

-   タブのタイトル
-   タブの URL
-   タブのファビコン
-   ウィンドウの位置とサイズ
-   タブグループ情報（名前、色）

#### 2. データ保存

すべてのデータは Chrome の `chrome.storage.local` API を使用して、**お客様のローカルデバイスにのみ保存**されます。

-   ✅ データはお客様のコンピュータに保存されます
-   ❌ データはリモートサーバーに**アップロードされません**
-   ❌ データは第三者と**共有されません**
-   ❌ この拡張機能はネットワークリクエストを**送信しません**

#### 3. データセキュリティ

すべてのデータがローカルに保存されるため、データセキュリティはお客様のデバイスのセキュリティに依存します。以下をお勧めします：

-   オペレーティングシステムとブラウザを最新の状態に保つ
-   強力なパスワードでコンピュータアカウントを保護する
-   公共のコンピュータで機密性の高いタブセッションを保存しないよう注意する

#### 4. サードパーティサービス

**Tab Session Lite はサードパーティサービスを一切使用しません**。これには以下が含まれますが、これらに限定されません：

-   分析ツール（Google Analytics など）
-   広告サービス
-   ユーザー追跡サービス
-   リモートデータベース

#### 5. 権限の説明

この拡張機能は以下の Chrome 権限を要求します：

| 権限             | 用途                                                      |
| ---------------- | --------------------------------------------------------- |
| `tabs`           | セッションを保存するためにタブのタイトルと URL を読み取る |
| `tabGroups`      | タブグループ情報の読み取りと復元                          |
| `storage`        | セッションデータをローカルに保存                          |
| `system.display` | ウィンドウ位置を正しく復元するための画面情報の取得        |

これらの権限はコア機能の実装にのみ使用され、他の目的には使用されません。

#### 6. 児童のプライバシー

この拡張機能は 13 歳未満の児童を対象としておらず、児童の個人情報を意図的に収集することはありません。

### Tab Session Lite 개인정보 처리방침

**Tab Session Lite**를 이용해 주셔서 감사합니다. 저희는 귀하의 개인정보를 매우 중요하게 생각합니다. 본 개인정보 처리방침은 귀하의 데이터를 어떻게 처리하는지 설명합니다.

#### 1. 데이터 수집

**Tab Session Lite는 어떠한 개인 데이터도 수집하지 않습니다.**

이 확장 프로그램은 다음 정보만 귀하의 기기에 로컬로 저장합니다:

-   탭 제목
-   탭 URL
-   탭 파비콘
-   창 위치 및 크기
-   탭 그룹 정보 (이름, 색상)

#### 2. 데이터 저장

모든 데이터는 Chrome의 `chrome.storage.local` API를 사용하여 **귀하의 로컬 기기에만 저장**됩니다.

-   ✅ 데이터는 귀하의 컴퓨터에 저장됩니다
-   ❌ 데이터는 원격 서버에 **업로드되지 않습니다**
-   ❌ 데이터는 제3자와 **공유되지 않습니다**
-   ❌ 이 확장 프로그램은 네트워크 요청을 **보내지 않습니다**

#### 3. 데이터 보안

모든 데이터가 로컬에 저장되므로 데이터 보안은 귀하의 기기 보안에 따라 달라집니다. 다음을 권장합니다:

-   운영 체제와 브라우저를 최신 상태로 유지
-   강력한 암호로 컴퓨터 계정 보호
-   공용 컴퓨터에서 민감한 탭 세션을 저장하지 않도록 주의

#### 4. 제3자 서비스

**Tab Session Lite는 제3자 서비스를 사용하지 않습니다**. 여기에는 다음이 포함되지만 이에 국한되지 않습니다:

-   분석 도구 (예: Google Analytics)
-   광고 서비스
-   사용자 추적 서비스
-   원격 데이터베이스

#### 5. 권한 설명

이 확장 프로그램은 다음 Chrome 권한을 요청합니다:

| 권한             | 용도                                                |
| ---------------- | --------------------------------------------------- |
| `tabs`           | 세션 저장을 위해 탭 제목과 URL 읽기                 |
| `tabGroups`      | 탭 그룹 정보 읽기 및 복원                           |
| `storage`        | 세션 데이터를 로컬에 저장                           |
| `system.display` | 창 위치를 올바르게 복원하기 위한 화면 정보 가져오기 |

이러한 권한은 핵심 기능 구현에만 사용되며 다른 목적으로는 사용되지 않습니다.

#### 6. 아동 개인정보

이 확장 프로그램은 13세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

#### 7. 정책 업데이트

본 개인정보 처리방침을 업데이트하는 경우, 이 페이지에 새 버전을 게시하고 "최종 업데이트" 날짜를 업데이트합니다.
