## English Version

### Tab Session Lite Privacy Policy

Thank you for using **Tab Session Lite**. We take your privacy seriously. This Privacy Policy explains how we handle your data.

#### 1. Data Collection

**Tab Session Lite does not collect any personal data.**

This extension only stores the following information locally on your device:
- Tab titles
- Tab URLs
- Tab favicons
- Window positions and sizes
- Tab group information (names, colors)

#### 2. Data Storage

All data is stored **exclusively on your local device** using Chrome's `chrome.storage.local` API.

- âœ… Data is stored on your computer
- âŒ Data is **NOT** uploaded to any remote servers
- âŒ Data is **NOT** shared with any third parties
- âŒ This extension does **NOT** make any network requests

#### 3. Data Security

Since all data is stored locally, data security depends on the security of your device. We recommend:
- Keeping your operating system and browser up to date
- Using strong passwords to protect your computer accounts
- Being cautious about saving sensitive tab sessions on public computers

#### 4. Third-Party Services

**Tab Session Lite does not use any third-party services**, including but not limited to:
- Analytics tools (such as Google Analytics)
- Advertising services
- User tracking services
- Remote databases

#### 5. Permissions Explained

This extension requests the following Chrome permissions:

| Permission | Purpose |
|------------|---------|
| `tabs` | Read tab titles and URLs to save sessions |
| `tabGroups` | Read and restore tab group information |
| `storage` | Store session data locally |

These permissions are used solely to implement core functionality and will not be used for any other purpose.

#### 6. Children's Privacy

This extension is not directed at children under 13 years of age, and we do not knowingly collect any personal information from children.

#### 7. Policy Updates

If we update this Privacy Policy, we will post the new version on this page and update the "Last Updated" date.

#### 8. Contact Us

If you have any questions about this Privacy Policy, please contact us through:
- GitHub Issues: [https://github.com/ben0588]
- Email: [energy9527z@gmail.com]
